# MRS_Lidar-based
ROS 2 framework for a Multi-Robot System (MRS) featuring Pioneer 3DX and Dobot CR3 with Unitree LiDAR 4D L1. Includes Bezier-based trajectory control for mobile navigation, robotic arm manipulation for pick-and-place, and automated volume estimation (cubing) for logistics. Integrated perception-to-action pipeline for autonomous handling.
